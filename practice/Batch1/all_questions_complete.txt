COMPREHENSIVE QUESTIONS FROM ALL 33 WHASTSAPP IMAGES

1. DIGITAL WALLET SYSTEM
- Design three classes: TransactionException, DigitalWallet, DigitalWalletTransaction
- TransactionException class:
  * Constructor: TransactionException(String errorMessage, String errorCode)
  * Method: String getErrorCode()
- DigitalWallet class:
  * Constructors: DigitalWallet(String walletId, String userName), DigitalWallet(String walletId, String userName, String userAccessCode)
  * Methods: getWalletId(), getUserName(), getUserAccessToken(), getWalletBalance(), setWalletBalance()
- DigitalWalletTransaction class:
  * Methods: addMoney(DigitalWallet digitalWallet, int amount), payMoney(DigitalWallet digitalWallet, int amount)
- Error codes and messages:
  * USER_NOT_AUTHORIZED: "User not authorized" (no access token)
  * INVALID_AMOUNT: "Amount should be greater than zero"
  * INSUFFICIENT_BALANCE: "Insufficient balance"
- Constraints: 1 ≤ numberOfWallets ≤ 100, 1 ≤ numberOfTransactions ≤ 2 × 10³

2. CRICKET AND FOOTBALL TEAMS
- Design Cricket and Football classes implementing Sport interface
- Cricket class:
  * Variable: int[] playerIDs (11 elements set to 1)
  * Methods: Cricket() constructor, calculateAvgAge(int[] age), retirePlayer(int id), playerTransfer(int fee, int id)
- Football class: Similar structure to Cricket class
- Requirements: Print messages for team formation, average age calculation, player retirement, and player transfer

- AbstractAccount base class with Account interface
- Concrete classes: CheckingAccount, SavingsAccount, CreditCardAccount
- BankingSystem class with methods:
  * createAccount(String accountType, String accountHolder, double creditLimit, double interestRate)
  * deposit(int accountNumber, double amount)
  * withdraw(int accountNumber, double amount)
  * checkBalance(int accountNumber)
  * makePayment(int cardNumber, double amount)
  * calculateInterest(int accountNumber)
- Custom exceptions for error handling
- Constraints: 1 ≤ n ≤ 100 (input lines)

4. SHAPE INHERITANCE
- Calculate area and perimeter of rectangle, circle, square using Shape interface
- Rectangle class: implements Shape, contains length and width
- Square class: extends Rectangle, sets length and width to side
- Circle class: implements Shape, contains radius
- Methods: getArea(), getPerimeter(), toString()
- Formulas: rectangle (length×width, 2×(length+width)), square (side², 4×side), circle (π×radius², 2×π×radius)
- Constraints: 1 ≤ length, width, side, radius ≤ 10²

  * performAll(T a, T b): Call all methods in order
- NumericDataType<T> methods:
  * addition: "Adding 2 generic instances", sum rounded to 2 decimals
  * subtraction: "Subtracting two generic instances", difference rounded to 2 decimals
  * multiplication: "Multiplying two generic instances", product rounded to 2 decimals
  * division: "Dividing two generic instances", quotient rounded to 2 decimals
  * performAll(T a, T b): Call all methods in order
- Constraints: numeric data 1 ≤ a,b ≤ 10⁹; strings 1 ≤ length ≤ 100
