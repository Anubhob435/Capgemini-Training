A subarray is a continuous (connected) part of an array. The elements must be next to each other
and in the same order.
Example:
Array = [1, 2, 3, 4]
Valid Subarrays:
1 [1]
2 [2]
3 [3]
4 [4]
5 [1, 2]
6 [2, 3]
7 [3, 4]
8 [1, 2, 3]
9 [2, 3, 4]
10 [1, 2, 3, 4]
Invalid Subarrays:
1 [1, 3] (elements are skipped)
2 [2, 4] (not continuous)
3 [3, 1] (order changed)

Find the Sum of All Possible Subarrays
Given an integer array of size n, find the sum of elements of all possible contiguous subarrays of the array.
Sample Input: [1, 2, 3]
Sample Output: 20
