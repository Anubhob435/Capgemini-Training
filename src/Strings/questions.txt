QUESTION 1: Check if a String is an Anagram
-------------------------------------------------------------------------------
Write a program to check given String is an Anagram or not and return a suitable message.


Definition:
    An anagram is a word or phrase formed by rearranging the letters of another word or phrase, 
    typically using all the original letters exactly once. For example, "listen" and "silent" 
    are anagrams of each other because they use the same letters.

Algorithm:
    1. First, check if both lengths of the String same or not. If same, you need to write the 
       implementation the program otherwise say directly "Not an Anagram".
    2. Compare the two Strings of every character. If they match for every character, 
       the strings are anagrams.

Example:
    Strings: "listen" and "silent"
    
    If you observe the two Strings, what characters are present in the s1 String same 
    characters are present in the s2 String. So, these two are Anagrams.

QUESTION 8: Remove Adjacent Duplicate Characters
-------------------------------------------------------------------------------
Write a program to Remove Adjacent Duplicate characters In a Given String and return the 
final string after all such duplicate removals have been made. It can be proven that the 
answer is unique.

Example 1:
    Input:
        s = "abbacd"
    
    Output:
        "cd"
    
    Explanation:
        In "abbacd" we could remove "bb" since the letters are adjacent and equal, 
        and this is the only possible move. The result of this move is that the string 
        is "aacd", of which only "aa" is possible, so the final string is "cd".

Example 2:
    Input:
        s = "accddaf"
    
    Steps:
        step1: addaf
        step2: aaf
        step3: f
    
    Output:
        "f"

QUESTION 11: Interchange Matrix Diagonals
-------------------------------------------------------------------------------
Given a square matrix of order n*n, you have to interchange the elements of both diagonals.

Explanation:
    Idea behind interchanging diagonals of a square matrix is simple. Iterate from 0 to n-1 
    and for each iteration you have to swap a[i][i] and a[i][n-i-1].

Example:
    Input:
        matrix[][] = {1, 2, 3,
                      4, 5, 6,
                      7, 8, 9}
    
    Output:
        matrix[][] = {3, 2, 1,
                      4, 5, 6,
                      9, 8, 7}
